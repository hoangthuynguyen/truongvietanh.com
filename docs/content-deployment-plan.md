# QUY TRÌNH SẢN XUẤT MỖI BÀI

```
1. Keyword research + brief (SEO Specialist) — 1 ngày
2. Viết draft (Writer) — 2-3 ngày
3. SEO review + internal link check (SEO) — 0.5 ngày
4. Edit + proofread (Editor) — 0.5 ngày
5. Nhập Directus + SEO fields + media (Editor) — 0.5 ngày
6. Review lần cuối + publish (Content Lead) — 0.5 ngày
7. Index request + social sharing — ngày publish
```

**Tổng lead time: 5-6 ngày/bài** (pipeline nhiều bài song song)
