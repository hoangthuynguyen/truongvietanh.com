### Kịch bản Assessment Day

```
08:30  Check-in → Quà chào mừng + Name tag + Bản đồ trường
09:00  Welcome Speech (Hiệu trưởng) — 10 phút
09:15  HS lớp 8–10 dẫn Tour bằng tiếng Anh (15 phút/group)
09:45  Assessment: Đa trí tuệ (MN) / IELTS mini test (C2–C3)
10:30  Phụ huynh: Coffee Talk + Q&A với GV
        Học sinh: Tham gia lớp PDR hoặc STEM demo
11:00  Giám đốc Học thuật đọc kết quả + tư vấn lộ trình PDR cá nhân
11:30  Tư vấn tài chính: Edu-Finance + Trả góp 0%
12:00  Lunch (ăn trưa tại trường — trải nghiệm thực đơn)
```

### GHL Workflow cho Assessment Day

```
Trigger: Lead đăng ký Assessment Day qua form
→ Day 0: Auto confirm ZNS + Calendar invite + PDF chuẩn bị
→ Day -1: Reminder ZNS "Ngày mai gặp bé [Tên] tại [Campus]!"
→ Day 0 (sau event): Tag "Assessed" trong GHL Pipeline
→ Day +1: Gửi PDF kết quả khảo sát + lộ trình PDR đề xuất
→ Day +3: Gọi follow-up (Admissions Officer)
→ Day +7: ZNS ưu đãi Early Bird (nếu chưa enroll)
→ Day +14: Sales Manager escalation (nếu chưa chốt)
```

### GHL Automation Workflows chi tiết

**Workflow 1: New Lead → 15-minute Response**
```
Trigger: New contact via any source (Form, Zalo, FB, Website)
├─ Step 1: Create opportunity in Pipeline "Tuyển sinh 2026"
├─ Step 2: Auto-tag source (utm_source, utm_medium, utm_campaign)
├─ Step 3: Send ZNS "Chào mừng [Tên PH]! Đây là thông tin HP Việt Anh: [link]"
├─ Step 4: Round Robin assign to Admissions Officer (by campus)
├─ Step 5: Internal notification → AO nhận alert trên app GHL
├─ Step 6: If no call within 15 min → Alert Sales Manager
└─ Step 7: If no call within 30 min → Escalate to RevOps Manager
```

**Workflow 2: Zalo Nurture 10-Day Drip**
```
Trigger: Contact enters Pipeline Stage "Khám phá"
├─ Phút 1: PDF HP + GT chương trình + link IELTS 6.0
├─ Ngày 2: Mời Assessment Day gần nhất + Calendar link
├─ Ngày 4: Video HS thuyết trình PDR bằng tiếng Anh
├─ Ngày 6: Testimonial PH cùng phường mới
├─ Ngày 7: AO gọi trực tiếp (task auto-created)
├─ Ngày 10: Reminder PDR + Early Bird ưu đãi
└─ Ngày 14: Nếu chưa book → chuyển sang "Warm Nurture" (monthly ZNS)
```

**Workflow 3: Assessment Day → Enrollment**
```
Trigger: Attended Assessment Day (tag: "assessed")
├─ Day +1: Email/ZNS kết quả khảo sát + PDF lộ trình PDR
├─ Day +3: AO gọi follow-up, đề xuất gói tài chính
├─ Day +5: ZNS testimonial PH đã enroll cùng đợt
├─ Day +7: Ưu đãi có thời hạn (voucher đồng phục/sách)
├─ Day +10: Sales Manager gọi nếu chưa quyết định
├─ Day +14: Final offer + deadline
└─ Day +21: Move to "Lost" nếu không phản hồi → Long-term nurture
```

**Workflow 4: Enrolled → Retention & Referral**
